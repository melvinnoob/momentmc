1. Pay Invoice by Credit

Endpoint: POST https://bill.momentmc.com/api/client/v1/invoices/{invoice_id}/pay

{
    "payment_method": "credits"
    }
curl --request POST \  
    --url https://bill.momentmc.com/api/client/v1/invoices/724/pay \  
    --header 'Authorization: Bearer YOUR_API_KEY' \  
    --header 'Content-Type: application/json' \  
    --data '{"payment_method": "credits"}'

2. Browse Products

Endpoint: GET https://bill.momentmc.com/api/website/v1/products

curl -X GET "https://bill.momentmc.com/api/website/v1/products" \  
    -H "Accept: application/json" \  
    -H "Authorization: Bearer YOUR_API_KEY"

3. Browse Categories

Endpoint: GET https://bill.momentmc.com/api/website/v1/categories

curl -X GET "https://bill.momentmc.com/api/website/v1/categories" \  
    -H "Accept: application/json" \  
    -H "Authorization: Bearer YOUR_API_KEY"
